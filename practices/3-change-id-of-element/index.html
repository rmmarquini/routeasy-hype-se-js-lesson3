Alterar o id de um elemento

Neste cenário, o código existente espera um clique no botão. Quando o botão é clicado, a função changeInput é acionada. changeInput tenta selecionar um campo de entrada com id inputEl. Porém, o campo de entrada existente não possui esse id. Edite o código Javascript para adicionar o id inputEl ao campo de entrada existente.

Confira seu código clicando no botão. 

// Seu código vai aqui
// ...

const button = document.querySelector('.wrapper button')

const changeInput = () => {
  const input = document.querySelector('#inputEl');
  if(input) {
    console.log(input)
    input.value = 'Hello World'
  }
}

button.addEventListener('click', changeInput)
