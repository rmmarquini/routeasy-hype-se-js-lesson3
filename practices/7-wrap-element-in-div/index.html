Envolver elemento em uma div

O código JavaScript abaixo prepara um eventListener em uma variável btn. Um elemento filho atribuído como um botão descende de outro elemento com id wrapper. À este botão foi atribuído à variável btn.
Porém, o elemento de botão existente não possui nenhum nó pai com wrapper de id.
Faça o código funcionar criando um novo elemento com id wrapper que envolve o elemento de botão existente.

Dica: talvez seja necessário usar os seguintes métodos: querySelector, createElement, setAttribute, append.

Confira seu código clicando no botão. 

// seu código aqui


// se espera que btn seja filho do elemento com id 'wrapper'
const btn = document.querySelector('#wrapper button');

if(btn) {
  const handleClick = () => {
    btn.innerText = 'Thank you! ❤️'
  }
  btn.addEventListener('click', handleClick)
}
